Detailed Description

template<uint32_t N>
struct EquidistantPWL< N >

Piecewise linear interpolation of look-up-table (LUT) values.

LUT values representing function values starting with y(x=in_fsr_lower) and ending with y(x=in_fsr_upper).

y-values of the LUT must correspond to equidistant X-axis points.

This version has run-time settable input value range.
